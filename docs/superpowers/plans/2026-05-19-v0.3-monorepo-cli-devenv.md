# v0.3 Monorepo, CLI Distribuível e Ambiente de Desenvolvimento — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Transformar vault-seed em monorepo incremental com `@dgk/cli` como primeiro pacote distribuível, devcontainer PT-BR portátil e setup local multiplataforma.

**Architecture:** A raiz vira workspace root (`private: true`) com `pnpm-workspace.yaml` apontando para `packages/*`. O vault permanece intacto na raiz. O CLI em `packages/cli` é um invólucro sem dependências de runtime sobre os scripts existentes. A fronteira do template é gerenciada via `package.template.json` e `pnpm-lock.template.yaml` que substituem os equivalentes do workspace durante `initialize.yml`.

**Tech Stack:** Node.js ≥22, pnpm workspaces, `node:util parseArgs`, `node:test`, `node:child_process`, `@changesets/cli`, devcontainers features (node + uv + common-utils), fnm (local Node version mgmt), uv (local Python tool mgmt).

**Spec:** `docs/superpowers/specs/2026-05-19-v0.3-monorepo-cli-devenv-design.md`

```js
import { spawn } from 'node:child_process';

export function run(cmd, args, opts = {}) {
  return new Promise((resolve, reject) => {
    const child = spawn(cmd, args, { stdio: 'inherit', shell: false, ...opts });
    child.on('close', code => {
      if (code !== 0) reject(new Error(`'${cmd} ${args.join(' ')}' exited with code ${code}`));
      else resolve();
    });
    child.on('error', reject);
  });
}
```

- [ ] **Step 3.6: Criar packages/cli/src/index.js**

```js
#!/usr/bin/env node
import { fileURLToPath } from 'node:url';
import { validate } from './commands/validate.js';
import { lint } from './commands/lint.js';
import { setup } from './commands/setup.js';
import { release } from './commands/release.js';
import { check } from './commands/check.js';

const COMMANDS = { validate, lint, setup, release, check };

export function resolveCommand(name) {
  return name in COMMANDS ? name : null;
}

if (process.argv[1] === fileURLToPath(import.meta.url)) {
  const [,, command, ...rest] = process.argv;

  if (!resolveCommand(command)) {
    console.error(`dgk: comando desconhecido '${command ?? ''}'`);
    console.error(`Uso: dgk <comando>`);
    console.error(`Comandos: ${Object.keys(COMMANDS).join(', ')}`);
    process.exit(1);
  }

  COMMANDS[command](rest).catch(err => {
    console.error(`dgk: ${err.message}`);
    process.exit(1);
  });
}
```

#!/bin/bash
# scripts/setup_node.sh
# Configura o ambiente Node.js — fnm preferido (cross-platform), nvm como fallback
set -e

setup_node_version() {
    echo "Configurando ambiente Node.js..."

    if command -v fnm &>/dev/null; then
        echo "fnm encontrado."
        eval "$(fnm env --use-on-cd)"
        fnm use --install-if-missing
        echo "Node.js $(node -v) ativado via fnm."

    elif [ -s "$HOME/.nvm/nvm.sh" ]; then
        echo "[AVISO] nvm encontrado. Recomendamos migrar para fnm (cross-platform)."
        echo "Instale fnm:"
        echo "  Windows : winget install Schniz.fnm"
        echo "  macOS   : brew install fnm"
        echo "  Linux   : curl -fsSL https://fnm.vercel.app/install | bash"
        \. "$HOME/.nvm/nvm.sh"
        [ -s "$HOME/.nvm/bash_completion" ] && \. "$HOME/.nvm/bash_completion"
        nvm use --install-if-missing

    else
        echo "[ERRO] Nenhum gerenciador de versão Node encontrado."
        echo "Instale o fnm:"
        echo "  Windows : winget install Schniz.fnm"
        echo "  macOS   : brew install fnm"
        echo "  Linux   : curl -fsSL https://fnm.vercel.app/install | bash"
        exit 1
    fi

    if ! command -v corepack &>/dev/null; then
        echo "[ERRO] corepack não encontrado. O Corepack acompanha versões modernas do Node.js."
        exit 1
    fi
    corepack enable
    corepack prepare --activate

    if ! command -v pnpm &>/dev/null; then
        echo "[ERRO] pnpm não encontrado após ativar o Corepack."
        exit 1
    fi
    echo "pnpm $(pnpm --version) encontrado."

    if [ ! -d "node_modules" ]; then
        echo "Dependências não instaladas. Rodando 'pnpm install'..."
        pnpm install --frozen-lockfile
    else
        echo "Dependências já instaladas."
    fi

    echo "Configuração do ambiente Node.js concluída."
}
