# PKM com Obsidian e VS Code

Gerenciamento de conhecimento pessoal (PKM) pode usar ferramentas diferentes. O vault-seed suporta dois editores principais: Obsidian e VS Code com Foam.

## Obsidian

Obsidian usa o vault como pasta local. Plugins como Dataview, Templater e Bases adicionam funcionalidades de banco de dados e automação diretamente sobre arquivos Markdown.

## VS Code com Foam

Foam é uma extensão do VS Code que adiciona navegação por links wiki, backlinks e visualização de grafo. Para quem já vive no VS Code, é uma entrada natural.

## Interoperabilidade

Os dois editores leem os mesmos arquivos. Links no formato `[[Nome da Nota]]` funcionam em ambos. O frontmatter YAML é compatível. A escolha de editor não afeta o vault.
