# Links

Links conectam notas dentro do vault. No Obsidian, o formato mais comum é o wikilink:

```markdown
[[Nome da Nota]]
```

Use links quando uma nota ajudar a explicar, complementar, contrastar ou continuar outra. Com o tempo, essas conexões viram o mapa real do seu conhecimento.

## Bons usos

- Conectar uma nota diária a um projeto ativo.
- Ligar um conceito a uma explicação mais detalhada.
- Apontar uma ferramenta para o guia que ensina a usá-la.
- Criar trilhas de leitura entre recursos relacionados.

## Links antes de pastas perfeitas

Uma nota só pode morar em uma pasta por vez, mas pode apontar para muitos lugares. Por isso, links são melhores que pastas para representar relações entre ideias. A pasta responde “onde isto vive agora?”; o link responde “com o que isto conversa?”.

Quando um grupo de links começar a aparecer com frequência, crie um [[O que são MOCs (Mapas de Conteúdo)|MOC]] para transformar a rede em uma porta de entrada legível.
