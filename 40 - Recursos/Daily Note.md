# Daily Note

Daily Note é uma nota criada para um dia específico. Ela funciona bem para captura rápida, planejamento do dia, revisão e registro de pequenas decisões que ainda não merecem uma nota própria.

Neste vault, notas diárias combinam com a pasta `10 - Diário`. Use essa pasta como uma caixa de entrada com data: algo aconteceu hoje, uma ideia apareceu durante uma reunião, ou você quer lembrar por que tomou uma decisão.

## Quando usar

Use uma Daily Note para:

- registrar tarefas e compromissos do dia;
- capturar ideias antes de decidir onde elas pertencem;
- anotar pequenas decisões operacionais;
- criar links para projetos, áreas ou recursos que foram tocados naquele dia.

## Como evoluir a nota

A Daily Note não precisa virar arquivo permanente para tudo. Depois da revisão, mova trechos importantes para notas mais estáveis em `20 - Projetos`, `30 - Áreas` ou `40 - Recursos`. O diário preserva o contexto; as notas permanentes preservam conhecimento reutilizável.

Comece pelo [[Template - Nota Diaria]] e conecte a nota diária com [[Links]] sempre que uma ideia apontar para outro lugar do vault.
