# Templater

Templater é um plugin da comunidade do Obsidian que cria notas a partir de modelos mais poderosos que o plugin core Templates. Ele é útil quando um modelo precisa inserir datas, caminhos, variáveis ou pequenos trechos dinâmicos.

Você não precisa dele para começar. O fluxo inicial do vault privilegia o plugin core Templates porque ele é mais simples e já vem com o Obsidian. Templater entra quando você quer automatizar detalhes repetitivos sem transformar a escrita em programação.

## Quando vale usar

Use Templater para:

- preencher datas automaticamente;
- criar títulos ou aliases a partir do nome do arquivo;
- inserir blocos condicionais;
- padronizar notas recorrentes;
- reduzir cópia manual em templates avançados.

## Cuidado operacional

Templates muito inteligentes podem ficar difíceis de manter. Prefira começar com modelos simples em `90 - Modelos` e só adicionar lógica quando houver repetição real. Se um template for essencial para usuários iniciantes, documente também uma alternativa com o plugin core Templates em [[Usando o Plugin Templates]].
