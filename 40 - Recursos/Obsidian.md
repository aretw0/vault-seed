# Obsidian

Obsidian é um aplicativo para trabalhar com notas Markdown locais. Ele valoriza links entre notas, visualização em grafo e extensões por plugins.

Neste vault, ele é a experiência principal para leitura, escrita e exploração visual do conhecimento. Como os arquivos continuam sendo Markdown comuns, você não fica preso ao aplicativo: pode editar as mesmas notas no [[VS Code]], versionar com Git e publicar como site.

## Por que usar neste template

- Mantém as notas em arquivos locais.
- Suporta wikilinks, backlinks e grafo visual.
- Funciona bem com templates, propriedades e plugins.
- Permite alternar entre escrita pessoal e publicação.

## O que configurar primeiro

Comece abrindo a pasta do vault no Obsidian. Depois, revise [[Plugins Essenciais e Recomendados]] e ative apenas o que fizer sentido para seu uso. O plugin core Templates já cobre o fluxo inicial; plugins comunitários como [[Templater]] podem ser adicionados depois.

Use o Obsidian para pensamento e navegação. Use automações e validações quando quiser garantir que o vault continue publicável e consistente.
